About This Website

I made this website to gain experience with full-stack development, particularly with Django, a Python web framework, as well as Materialize, a responsive front-end framework based on Material Design. This website is hosted on an Ubuntu server I set up at home with Gunicorn and Nginx, and much of the content in this website is dynamically generated from a PostgreSQL database (for educational purposes). I made this website using the following frameworks and libraries:

  • Django
  • jQuery
  • Materialize
  • Font Awesome
  • Typed.js
  • Particles.js
  • ScrollReveal.js
  • Velocity.js

If you have any suggestions or comments, please email me at czhao39@gmail.com. Please note that Materialize is still in alpha so there may be bugs in the UI (let me know if you notice any!).
